# Build Plans

This folder contains the structured build plan for the project. Each plan is a self-contained sprint document with phases, tasks, and executable prompts for our coding agent.

Plans are numbered in execution order. Complete each plan before moving to the next.

---

## Plan Structure

Every plan follows this format:

```
# Plan Title

## Overview
What this plan accomplishes and its success criteria.

## Phases
### Phase X: [Name]
#### Task X.1: [Task Name]
- **Description:** What needs to be built
- **Agent Prompt:** Full copy-paste prompt for the coding agent
- **Acceptance Criteria:** How to verify the task is complete
- **Dependencies:** What must be done before this task
```
